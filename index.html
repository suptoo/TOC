Study Guide: Theory of Computation & Automata

A comprehensive review based on past exam questions from 2019, 2021, and 2023.

1. Finite Automata (FA)

This is a fundamental topic. You need to understand the different types of finite automata, their applications, how to design them, and how to convert between them.

DFA vs. NFA

Know the formal definitions of both. The key difference is that in a DFA, for each state and input symbol, there is exactly one next state, whereas in an NFA, there can be zero, one, or multiple next states.

  • Differentiate between DFA and NFA with a suitable example. (2021)
  • Between DFA and NFA, which one is better for handling a huge number of states? Justify your answer. (2021)
  • Differentiate between DFA and NFA. Which one is better when there are large numbers of states? (2019)

Designing DFAs and NFAs

  • Design a DFA that models a security system activating an alarm if a binary sequence contains an odd number of 0's and an odd number of 1's. (2023)
  • Design an NFA to detect a fault pattern represented by all binary strings that contain `010` or `101` as a substring. (2023)

2. Regular Expressions (RE) & Regular Languages

Writing and Simplifying Regular Expressions

  • Find a regular expression for the language of all strings over `{a, b}` that do not end with `01`. (2023)
  • Simplify the expression: `(a*b)|(ba*b)|(b.ab*)`. (2023)

3. Context-Free Grammars (CFG) & Languages

Designing and Understanding CFGs

  • Design a CFG for the language `L = {ww^R | w is in (0+1)*}` (palindromes). (2023)

Ambiguity

  • For a given grammar and the expression `x1y+x*y`, provide the leftmost and rightmost derivations and explain the reason for the ambiguity. (2023)

4. Pushdown Automata (PDA)

Designing PDAs

  • Design a PDA for `L = {a^i b^j c^k | i,j,k >= 0, and i=j or i=k}`. (2021)

5. Turing Machines (TM)

Designing Turing Machines

  • Construct a Turing machine that accepts the language `{a^n b^n c^* d^n | n >= 0}`. (2023)
